**Comandos Unix:**

ls- lista

mkdir - cria pasta 
cd - Entra na pasta 
cd .. - Sai da pasta 
rm -r - deleta pasta 
cls - limpa o terminal 
cp - copia o arquivo 
mv - mover arquivos 
pwd - mostra o path atual 
cat - abre o arquivo
