Comandos Unix:
==================

+ **ls** - lista todos os arquivos da pasta
+ **ls -al** - lista todos os arquivos e arquivos ocultos
+ **mkdir** - cria pasta
+ **cd** - Entra na pasta
+ **cd ..** - Sai da pasta
+ **rm -r** - deleta pasta
+ **cls** - limpa o terminal
+ **cp** - copia o arquivo
+ **mv** - mover arquivos
+ **pwd** - mostra o path atual
+ **cat** - abre o arquivo informado
---

Comandos GIT:
=================

+ **git init** - inicia o Git
+ **git config** --global http.proxy http://10.1.118.254:3128
+ **git commit -m "identificação do comit - Comente o que você adicionou/alterou"**
+ **git add (arquivo)** - Atualiza um git
+ **git status** - informa o que foi alterado
+ **git push** - Salva as atualizações para o Branch atual
+ **git diff**
+ **git clone url_do_repositorio_externo_ou_local**
+ **git branch** - lista as branches activas
+ **git branch -D nome_da_branch** - Deleta a branch
+ **git checkout -B** - cria uma branch
+ **git merge nome_da_branch** - Anexa uma branch dentro de outra
